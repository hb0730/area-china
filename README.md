# Area-China

中华人民共和国行政区划：省级（省份直辖市自治区）、 地级（城市）、 县级（区县）

## 数据格式

数据格式为`json`格式

```json
[
  {
    "code ": "编码",
    "name": "名称",
    "children": [ 
      {
        "code": "编码",
        "name": "名称",
        "children": []
      }
    ]
  }
]
```
